Methods

Given this method for the following questions:

public static class Homework
{
    public static string Pluralize(string text, int quantity)
    {
        if (quantity == 1)
        {
            return text;
        }
        
        if (text.EndsWith("s"))
        {
            return text + "es";
        }
        return text + "s";
    }
}

  1. What ultimately is the value of x? 

    var x = Homework.Pluralize("Basket", 2);
    1. Basket
    2. Baskets
    3. Basketes
    4. Nothing

  2. What ultimately is the value of x? 

    var y = 1;
var str = "Bus"
var x = Homework.Pluralize(str, y);
    1. Buss
    2. Bus
    3. Buses
    4. Nothing

  3. What ultimately is the value of stuff? 

    var str = "Soup";
var stuff = "";
for (var x = 0; x <= 2; x++)
{
    // the "+=" operator is shorthand for "myVar = myVar + somethingElse"
    stuff += x;
    stuff += " ";
    stuff += Homework.Pluralize(str, x);
    stuff += " ";
}
    1. 0 Soups 1 Soup 2 Soups
    2. 0 Soups 1 Soup
    3. 1 Soup 2 Soups 3 Soups
    4. 0Soups 1Soup 2Soups
