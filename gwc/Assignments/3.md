## Kitchen Sink

20. What will be the final value of `age`?

    ``` js
    const age = 0;
    const birthYear = 1907;
    const currentYear = 1963;
    const hasHadBirthday = false;
 
    while (birthYear < currentYear)
    {
        age++;
        currentYear--;
    }
 
    if (hasHadBirthday === false)
    {
        age--;
    }
    ```

    > 54
    > !55
    > 56
    > -1
